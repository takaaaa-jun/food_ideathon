## [1.0.0] - 2025-10-09

### 追加
- 材料名からレシピを検索する基本機能
- OR検索、AND検索、NOT検索の3つの検索モード
- レシピ詳細表示（材料、手順、調理時間、人数）
- レシピ全体および各材料の栄養素表示機能
- 基礎レシピ検索機能
- Docker Compose環境の構築
- Nginx リバースプロキシの設定
- MySQL データベース連携
- レスポンシブデザイン対応

### 技術的な実装
- Flask アプリケーションの構築
- MySQL との連携（mysql-connector-python）
- Gunicorn による本番環境対応
- Docker マルチコンテナ構成（app, db, reverse-proxy）
- データベース栄養素テーブルとの結合クエリ
- normalized_name = 'null' のレシピ除外ロジック

### インフラ
- Docker Compose による環境構築
- Nginx リバースプロキシ設定
- MySQL 8.0 データベース
- 環境変数による設定管理

## 変更履歴の書き方（ガイド）

### バージョン番号のルール
- **メジャーバージョン (X.0.0)**: 大きな機能追加や破壊的変更
- **マイナーバージョン (0.X.0)**: 新機能追加（後方互換性あり）
- **パッチバージョン (0.0.X)**: バグ修正や小さな改善

### カテゴリ
- **追加 (Added)**: 新機能
- **変更 (Changed)**: 既存機能の変更
- **非推奨 (Deprecated)**: 近い将来削除される機能
- **削除 (Removed)**: 削除された機能
- **修正 (Fixed)**: バグ修正
- **セキュリティ (Security)**: セキュリティ関連の修正

### 記入例

```markdown
## [1.1.0] - 2025-11-22

### 追加
- 検索結果のページネーション機能
- レシピのお気に入り登録機能

### 変更
- 検索結果の表示順序を関連度順に変更
- UIデザインをモダンなスタイルに更新

### 修正
- 栄養素計算の小数点以下の表示エラーを修正
- 検索モード切り替え時の状態保持の不具合を修正
```
