## [1.0.0] - 2025-10-09

### 追加
- 材料名からレシピを検索する基本機能
- OR検索、AND検索、NOT検索の3つの検索モード
- レシピ詳細表示（材料、手順、調理時間、人数）
- レシピ全体および各材料の栄養素表示機能
- 基礎レシピ検索機能
- Docker Compose環境の構築
- Nginx リバースプロキシの設定
- MySQL データベース連携
- レスポンシブデザイン対応

---

### 技術的な実装
- Flask アプリケーションの構築
- MySQL との連携（mysql-connector-python）
- Gunicorn による本番環境対応
- Docker マルチコンテナ構成（app, db, reverse-proxy）
- データベース栄養素テーブルとの結合クエリ
- normalized_name = 'null' のレシピ除外ロジック

### インフラ
- Docker Compose による環境構築
- Nginx リバースプロキシ設定
- MySQL 8.0 データベース
- 環境変数による設定管理
